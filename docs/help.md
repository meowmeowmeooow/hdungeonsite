# 🆘 Help & Support

Need assistance? Here is how to get the help you need.

---

## ❓ Frequently Asked Questions

??? question "How can I see the server?"
    Check out the `#rules` channel, which will contain instructions on how to see all the channels.

??? question "Why can't I send GIFs?"
    Due to spam, GIFs are limited to new members. By levelling up (or boosting), you unlock the ability to use GIFs and play games. Level 3 is the level needed to send gifs.

??? question "How can I get more colours and roles?"
    Levelling up!

??? question "How can I level up?"
    You can level up fastest by posting media (images, videos, and GIFs), or by speaking in `#lounge`.

## All Commands

### 💰 Economy
Commands for managing your Pokécoins and playing games of chance.

| Command | Description |
| :--- | :--- |
| `+balance` | Check your current Pokécoin balance. |
| `+income` | Work your job to earn coins (6-hour cooldown). |
| `+steal <@user>` | Attempt to steal from another user (24-hour cooldown). |
| `+slots` | Play the slot machine. |
| `+coinflip <amount> <heads/tails>` | Bet on a coin flip. |
| `+roulette <amount> <bet>` | Bet on roulette (red, black, or numbers 0-36). |
| `+pay <@user> <amount>` | Transfer coins to another user. |
| `+shop [item_number]` | View the shop or buy an item (e.g., Pokéballs). |
