# Help & Support

Need assistance? Here is how to get the help you need.

### Frequently Asked Questions

**Q: How can I see the server?**
Check out the `#rules` channel, which will contain instructions on how to see all the channels.

**Q: Why can't I send GIFs?**
Due to spam, GIFs are limited to new members. By levelling up (or boosting), you unlock the ability to use GIFs and play games.

**Q: How can I get more colours and roles?**
Levelling up!

**Q: How can I level up?**
You can level up fastest by posting media (images, videos and GIFs), or speaking in `#lounge`.

### Contacting Staff
If you have a serious issue (harassment, bugs, etc.), please **Open a Ticket** in the `#support` channel.
